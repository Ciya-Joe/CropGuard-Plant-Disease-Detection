# 🌱 CropGuard - AI Powered Plant Disease Detection

## 📌 Overview
CropGuard is a deep learning-based system that detects plant diseases from images of leaves, stems, and fruits using ResNet50.

## 🚀 Features
- Disease classification with confidence score
- Supports multiple plant parts
- User-friendly Streamlit interface
- Transfer learning for high accuracy

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- ResNet50
- Streamlit

## 📊 Model Details
- Input size: 224x224
- Architecture: ResNet50 + Custom Layers
- Techniques: Data Augmentation, Dropout, Early Stopping
